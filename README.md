# ♻️ Trash Sorting Game (V1)

An interactive, web-based recycling simulation built to practice **HTML5 Drag and Drop API** and dynamic **DOM manipulation**.

### 🎮 How to Play
- Select a waste item (Plastic Bottle, Paper, Glass Bottle, or Metal Can).
- Drag the item and drop it into the **Trash Bin**.
- Watch as the items are dynamically removed from the screen upon a successful drop!

### 🚀 Key Features
- **Interactive UX:** Visual feedback during drag-and-drop actions.
- **Vanilla JavaScript:** No external libraries or frameworks used; pure core logic.
- **Modern CSS:** Clean and responsive layout using Flexbox.
- **Global Standards:** Developed with English naming conventions and clean code principles.

### 🛠️ Technical Implementation
- **PreventDefault:** Overrode default browser behavior to enable custom drop zones.
- **Event Listeners:** Implemented `dragstart`, `dragover`, and `drop` events.
- **DOM Management:** Used `element.remove()` to handle item lifecycle.
